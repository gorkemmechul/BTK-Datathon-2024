# BTK Academy x Google Datathon 2024

Started this with a group but ended up finishing it solo. The competition ran on Kaggle for about a week (September 9–15, 2024), organized by BTK Academy, Google, and Girişimcilik Vakfı.

## The task

Predict evaluation scores for ~11,000 Girişimcilik Vakfı applicants from 2023, using historical application data going back to 2014. Features included anonymized university, family, and demographic info. Metric was RMSE.

## What I did

Explored the dataset, did some feature engineering, trained a regression model, and submitted predictions via Kaggle. Two successful submissions, both evaluated.

Not the most complex pipeline, but it was my first proper end-to-end ML competition and it got me comfortable with the Kaggle workflow.

## Files

- `datathon_2024_2.csv` — final submission

## Tools

Python, Pandas, Scikit-learn, Kaggle
